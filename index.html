    学好C语言：学习的是思维逻辑，而不是简单的语法

    C语言特征

    编程泛式：所支持的程序实现模式。

    C语言是面向过程的一门编程语言。

    C++支持四种编程泛式的语言：面向过程、面向对象、泛型编程、函数式编程。

    输出函数printf

    头文件： stdio.h

    原型：int printf(const char *format, ...);

    format ： 格式控制字符串 ；… ：可变参数列表； 返回值：输出字符的数量。

    返回值为int，例子：如果n = 123;，则返回为3，若遇到回车，即'\n'（这是一个字符），则返回值为4。

    输入函数scanf

    头文件： stdio.h

    原型：int scantf(const char *format, ...);

    format ： 格式控制字符串 ；… ：可变参数列表； 返回值：成功读入的参数个数。

    返回值可能为正整数、0（scanf("hello");）。负数（不合法），常用于while判断条件中（EOF = -1，End of file）。

    基本运算符

    运算符 说明 例子
    = 赋值运算符 a = b;
    +、-、*、/、（） 基本四则运算 a = (b + c) * d;
    % 求余运算符 a = b % 2;
    &、|、^、~ 位运算（很重要） a = ~b | c;
    <<、>> 左移、右移（补0） a = b >> 2;

    / ：两边都是整数，整除，向下取整。若左右有一边为浮点数，则会是小数。

    位 ： 比特位。64位操作系统，int占据32位（4个字节），8位一个字节。

    ^ ：异或，相同为0，不同为1。逆运算（减法是加法的逆运算）。异或是其本身的逆运算。例：a ^ b = c <=> c ^ a = b <=> c ^ b = a。

    <<：左移一位，相当于 * 2。

    回文整数的判断

    系统内核中有两条宏语句：（逻辑归一化）

    #define likely(x) __builtin_expect(!!(x), 1)//x经常成立

    告诉CPU：x是经常成立还是不经常成立
