C语言特征

编程泛式：所支持的程序实现模式。

C语言是面向过程的一门编程语言。

C++支持四种编程泛式的语言：面向过程、面向对象、泛型编程、函数式编程。

输出函数printf

头文件： stdio.h

原型：int printf(const char *format, ...);

format ： 格式控制字符串 ；… ：可变参数列表； 返回值：输出字符的数量。

返回值为int，例子：如果n = 123;，则返回为3，若遇到回车，即'\n'（这是一个字符），则返回值为4。

输入函数scanf

头文件： stdio.h

原型：int scantf(const char *format, ...);

format ： 格式控制字符串 ；… ：可变参数列表； 返回值：成功读入的参数个数。

返回值可能为正整数、0（scanf("hello");）。负数（不合法），常用于while判断条件中（EOF = -1，End of file）。

如何查看文件权限

ls -al 文件名称

printf家族：sprintf,fprintf

sprintf（字符串）：打印输出内容，向字符数组中打印值（打印位置不同）。主要作用于字符串拼接。

fprintf：打印到文件当中去。

本质功能没有区别，只是打印位置不同。

标准输出重定向

‘>’ 标准输出重定向符，./a.out > output 将标准输出打印到output文件中。其中，stderr是标准错误输出，无法打印到output中。（stdout可以打印）

按位与

n & 1：二进制下n的最低位置为1

n & 2：二进制下n的倒数第二位位置为1

基本运算符

运算符 说明 例子
= 赋值运算符 a = b;
+、-、*、/、（） 基本四则运算 a = (b + c) * d;
% 求余运算符 a = b % 2;
&、|、^、~ 位运算（很重要） a = ~b | c;
<<、>> 左移、右移（补0） a = b >> 2;

/ ：两边都是整数，整除，向下取整。若左右有一边为浮点数，则会是小数。

位 ： 比特位。64位操作系统，int占据32位（4个字节），8位一个字节。

^ ：异或，相同为0，不同为1。逆运算（减法是加法的逆运算）。异或是其本身的逆运算。例：a ^ b = c <=> c ^ a = b <=> c ^ b = a。

<<：左移一位，相当于 * 2。

数学库函数

pow函数：指数函数

头文件：math.h

原型：double pow(double a, double b); a是底数，b是指数，返回值是计算结果。

sqrt函数：开平方函数

头文件：math.h

原型：double sqrt(double x);x为被开方数，返回值为计算结果。

ceil函数：上取整函数

头文件：math.h

原型：double ceil(double x);

ceil(4.5) = 5

floor函数：上取整函数

头文件：math.h

原型：double floor(double x);

floor(4.9) = 4

abs函数：整数绝对值函数

头文件：stdlib.h

原型：int abs(int x);

fabs函数：实数绝对值函数

头文件：math.h

原型：double fabs(double x);

log函数：以e为底对数函数

头文件：math.h

原型：double log(double x);

log10函数：以10为底对数函数（利用换底公式）

头文件：math.h

原型：double log10(double x);

acos函数：三角函数

头文件：math.h

原型：double acos(double x); x为角度的弧度制。

acos(-1) = \Pi
